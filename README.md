# fullstack_developer_capstone

This is the capstone project for the Full Stack Developer certification.

## Project Name
fullstack_developer_capstone

## Repository Name
fenil-fullstack_developer_capstone

## Application Components
1. **Django Backend**: Controls the application logic and serves the frontend.
2. **Node.js Microservice**: Handles dealership data and reviews.
3. **Sentiment Analysis**: A Flask service to analyze review sentiment.
